## What this is

My academic website. A single HTML file. Serif type, a grain overlay, a progress bar that shifts color as you scroll, and enough whitespace to think.

Live at → **[pietromanganelliconforti.github.io/website](https://pietromanganelliconforti.github.io/website/)**

---

## Stack

```
HTML  ──  one file
CSS   ──  custom properties, no preprocessor
JS    ──  vanilla, ~120 lines, no dependencies
Fonts ──  Newsreader (serif) · JetBrains Mono (mono)
```

That's it. Espresso?

---

## Features

- 🎨 **Scroll-tinted background** — the page slowly changes skin as you descend into the rabbit hole
- 🎯 **Section-aware accent** — progress bar and active nav link pick up a per-section color
- 🌙 **Dark mode** — respects `prefers-color-scheme`, persists to `localStorage`
- ✨ **Scroll reveal** — sections fade in via `IntersectionObserver`
- 📋 **Copy email** — click the address, it teleports straight to clipboard
- ⬆️ **Back to top** — appears after 300px of scroll, a rope to get out of the pit
